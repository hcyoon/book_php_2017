# 2-3. echo문의 대소문자 구분

source: `{{ page.path }}`

```php
<?
	echo "--------------------------------<br>";
	echo "사과<br>";
	echo "오렌지<br>";
	echo "--------------------------------<br>";
	ECHO "&^%$&*%&&%*&(*(<br>";
	ECHO "--------------------------------<br>";
	ECHO "사과<br>";
	ECHO "오렌지<br>";
	ECHO "--------------------------------<br>";
?>
```


결과 :
```
--------------------------------
사과
오렌지
--------------------------------
&^%$&*%&&%*&(*(
--------------------------------
사과
오렌지
--------------------------------
```
